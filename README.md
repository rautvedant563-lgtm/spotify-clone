Spotify Clone 🎵

A Spotify-inspired music player built using HTML, CSS, and JavaScript.

Features

- Spotify-like user interface
- Play/Pause controls
- Progress bar
- Responsive design

Technologies Used

- HTML
- CSS
- JavaScript

About

This project was created as a front-end web development learning project to practice HTML, CSS, JavaScript, DOM manipulation, and UI design.

Note

Most of the project functionality has been implemented. The audio playback feature is currently experiencing a technical issue, while the rest of the application is available for demonstration and learning purposes.
